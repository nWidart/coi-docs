# Setup

To get started, clone the following repository: https://github.com/MaFi-Games/Captain-of-industry-modding

# Set your `COI_ROOT` env variable

Set this to the location where the game is installed. Example: `C:/Steam/steamapps/common/Captain of Industry`
You can get this path from the Steam client via `Properties... -> Local Files -> Browse.`

# Follow setups on Github

The readme file of the  `Captain-of-industry-modding` repository contains all further instructions needed. Including setting up Unity development.
